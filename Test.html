log4veritas is exactly that: a collection of knowledge and experience accross many different topics and disciplines. This website is split into two parts. The first section, AMPC, holds information about the objects we create. The other section is BSQS, a vast library containing the theory and technical knowledge required to make the objects detailed in AMPC (plus a lot more).

The Archivum de Machinator, Propositium, et Creatio (AMPC), meaning The Archive of Engineering, Design, and Creation, is a complete guide to the objects we make. It is not a "how-to" guide, but more of a description of "what happened", as Adam Savage would say.

The Bibliotheca Scire Quod Sciendum, which roughly translates from latin to The Library of Knowlege Which is Worth Knowing, holds all the theory, the reference material, textbook solutions, automatic calculator, the list goes on. It is meant to be a treasure trove of information, always being updated whenever a new piece of insight is gained.

The information here is theoretical in nature. Best practices and techniques are usually described in AMPC; however, there may be some overlap between it and the Engineering section in BSQS. This is unfortunately unavoidable.

BSQS is organized first by identifying the overarching subject, and then branching off into the various sub-topics. Everything is in a place that makes sense. Sometimes the same file is listed under two separate sub-topics, and that's ok. It gives the library a more complete feel, as the characterization of a file is not always black and white. Inter-disiplinary projects are somewhat common, and so they should be available to the user no matter which component of the project they thought of first.
